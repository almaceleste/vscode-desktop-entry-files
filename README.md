# Desktop Entry files support

Adds support for .desktop and .directory files to the VS Code.

## Installation

Install from Github: []()  
or  
Launch VS Code Quick Open (Ctrl+P) and run command `ext install almaceleste.desktop-entry-files`

## Features

* Syntax highlighting
* Errors highlighting
